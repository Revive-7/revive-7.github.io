# Revive OS Website

Official website for the Revive OS project.

## Live Site

revive-os.github.io

## Overview

This is a static multi-page website built with plain HTML, CSS, and JavaScript.
It includes:

- Home page with project info and editions
- About page with team/member information
- Contact page with support/community links
- Light/Dark mode toggle with system-theme auto detection

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## Run Locally

No build step is required.

1. Open [index.html](./index.html) in your browser.

## Notes

- Update page links in `index.html`, `about.html`, and `contact.html` together when renaming files.
- Keep `sitemap.xml` in sync with public page URLs.
